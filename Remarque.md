# Les types de cours a HEI
- __Cours techniques__ : PROG, SYS, TH, WEB : C'est pour travailler avec la machine
- __Cours d'ouverture / transversaux__ : LV, MGT, PRO : C'est pour travailler avec les humains

## Qui est le plus important des deux ?
Logiquement, nous travaillons pour des humains donc les cours d'ouverture sont bcp plus important.
